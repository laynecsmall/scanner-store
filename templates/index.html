{% block content %} This is the scanner store interface, used to store and retrieve data from the SmartCities scanner device. Routes are as follows:
Endpoint Method Description
/ GET This page, description and help
/new/result/ POST send a new result. Post form should have:
    device_name
    time
    raw_results
/new/device/ POST Create a record for new device. Post form should have:
    device_name
    device_type
    sensor_x
    sensor_y
/get/results/latest_for_device/>device_name< GET returns the latest result submitted for >device_name<
/get/results/latest_n_for_device/>n</>device_name< GET returns the latest n results submitted for >device_name<
/get/device/details/>device< GET returns the details for >device<
/get/device/all GET returns the details for all registered devices
